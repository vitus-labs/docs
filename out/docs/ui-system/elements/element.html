Element

Element is the smallest possible unit to be used for creating components. By default it renders div. You can render any valid HTML tag by passsing it as a prop.Element is by default inline-flex. You can set flex by using boolean block value which will also stretch the element to width of 100%.Element also checks by default if tag passed in is type of inline HTML tag like button or span. If so, other helper sub-elements rendered inside (beforeContent and afterContent wrappers) are by default converted to span element so it is HTML valid.

Examples

Horizontal Element example

// import { Element } from '@vitus-labs/elements'


const BeforeAfter = () => <Element css={beforeAfterStyles} />


const Content = () => <Element css={contentStyles} />


const beforeAfterStyles = `
  ${defaultCss};
  width: 40px;
  height: 100%;
`


const contentStyles = `
  ${defaultCss};
  width: 100%;
  height: 100%;
`


const styles = `
  ${defaultCss};
  width: 100%;
  min-width: 320px;
  height: 240px;
  padding: 16px;
`


//  Element 
render(
  <Element
    gap={16}
    content={Content}
    css={styles}
    beforeContent={BeforeAfter}
    afterContent={BeforeAfter}
  />
)

Render children

To render children inside component you can pass content as children, label, or content prop. Label or content might be nicer or more comfortable for inline elements. If you pass more of them children will have the highest priority to be rendered and label or content will be ignored. Priorities are children content label. You can pick any of those prop names based on your prefeference.
// import { Element } from '@vitus-labs/elements'


const beforeAfterStyles = `
  ${defaultCss};
  width: 40px;
  height: 100%;
`


const contentStyles = `
  ${defaultCss};
  width: 100%;
  height: 100%;
`


const styles = `
  ${defaultCss};
  width: 100%;
  min-width: 240px;
  height: 48px;
  padding: 16px;
  margin-bottom: 12px;
`


const BeforeAfter = () => <Element css={beforeAfterStyles} />


const Content = () => <Element css={contentStyles} />


//  Element 
render(
  <>
    {/* used content here */}
    <Element
      gap={16}
      content={Content}
      css={styles}
      beforeContent={BeforeAfter}
      afterContent={BeforeAfter}
    />


    {/* used label here */}
    <Element
      gap={16}
      label={Content}
      css={styles}
      beforeContent={BeforeAfter}
      afterContent={BeforeAfter}
    />


    {/* used children here */}
    <Element
      gap={16}
      content={Content}
      css={styles}
      beforeContent={BeforeAfter}
      afterContent={BeforeAfter}
    >
      <Content />
    </Element>
  </>
)

Before & After content

You can pass beforeContent or afterContent. This may be also kind of alternative to using :before or :after pseudo elements. This is useful especially in cases like buttons with loading icon or input wrappers with icons or anything similar. It supports more types to render before/after content, check out examples.
// import { Element } from '@vitus-labs/elements'


const beforeAfterStyles = `
  ${defaultCss};
  width: 40px;
  height: 100%;
`


const contentStyles = `
  ${defaultCss};
  width: 100%;
  height: 100%;
`


const styles = `
  ${defaultCss};
  width: 100%;
  min-width: 240px;
  height: 48px;
  padding: 16px;
  margin-bottom: 12px;
`


const BeforeAfter = () => <Element css={beforeAfterStyles} />


const Content = () => <Element css={contentStyles} />


{
  /*
  beforeContent - renders box on the leftt side
  afterContent - renders box on the right side
*/
}


//  Element 
render(
  <>
    {/* [1] Function as prop */}
    <Element
      gap={16}
      content={Content}
      css={styles}
      beforeContent={BeforeAfter}
      afterContent={BeforeAfter}
    />


    {/* [2] Rendered component as prop */}
    <Element
      gap={16}
      content={Content}
      css={styles}
      beforeContent={<BeforeAfter />}
      afterContent={<BeforeAfter />}
    />


    {/* [3] Function as prop */}
    <Element
      gap={16}
      content={Content}
      css={styles}
      beforeContent={() => <BeforeAfter />}
      afterContent={() => <BeforeAfter />}
    />
  </>
)

Vertical & Horizontal alignment

Element can be aligned horizontally or vertically. So you can have beforeContent on the left side of children or above. The same applies for afterContent (below children or on the right side). Check out the examples below to see more.

Horizontal variant

Element aligns inner elements horizontally by default.
// import { Element } from '@vitus-labs/elements'


const beforeAfterStyles = `
  ${defaultCss};
  width: 40px;
  height: 100%;
`


const contentStyles = `
  ${defaultCss};
  width: 100%;
  height: 100%;
`


const styles = `
  ${defaultCss};
  width: 100%;
  min-width: 240px;
  height: 48px;
  padding: 16px;
  margin-bottom: 12px;
`


const BeforeAfter = () => <Element css={beforeAfterStyles} />


const Content = () => <Element css={contentStyles} />


{
  /* Element is horizontal by default */
}


//  Element 
render(
  <Element
    gap={16}
    content={Content}
    css={styles}
    beforeContent={BeforeAfter}
    afterContent={BeforeAfter}
  />
)

Vertical variant

For vertical aligning you can use prop vertical set to true or use direction prop which can be set to rows | inline | reverseRows | reverseInline.
// import { Element } from '@vitus-labs/elements'


const beforeAfterStyles = `
  ${defaultCss};
  height: 40px;
  width: 100%;
`


const contentStyles = `
  ${defaultCss};
  width: 100%;
  height: 100%;
`


const styles = `
  ${defaultCss};
  width: 50%;
  height: 320px;
  padding: 16px;
  margin-left: 16px;
  margin-right: 16px;
`


const BeforeAfter = () => <Element css={beforeAfterStyles} />


const Content = () => <Element css={contentStyles} />


//  Element 
render(
  <>
    {/* make it vertical by adding `vertical` prop */}
    <Element
      vertical
      gap={16}
      content={Content}
      css={styles}
      beforeContent={BeforeAfter}
      afterContent={BeforeAfter}
    />


    {/* make it vertical by adding `direction` prop */}
    <Element
      direction="rows"
      gap={16}
      content={Content}
      css={styles}
      beforeContent={BeforeAfter}
      afterContent={BeforeAfter}
    />
  </>
)
