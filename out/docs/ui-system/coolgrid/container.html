Container

Usage example

For enabling responsivity, wrap it up by Provider with configured theme
import { Provider, Container, Row, Col, theme } from '@vitus-labs/coolgrid'


export default () => (
  <Provider theme={theme}>
    <Container>
      <Row>...</Row>
      <Row>...</Row>
    </Container>
  </Provider>
)
Container doesn't have to be used with other coolgrid components only. Feel free to use Container even for wrapping other content.
import { Provider, Container, Row, Col, theme } from '@vitus-labs/coolgrid'


export default () => (
  <Provider theme={theme}>
    <Container>{/* some random components here */}</Container>
  </Provider>
)

Overriding defaults

You can even override cofiguration for a specific Container and its children. Container component accepts columns and width props.
import { Provider, Container, Row, Col, theme } from '@vitus-labs/coolgrid'


// defined breakpoints in Provider
const breakpoints = {
  xs: 320,
  sm: 768,
  md: 1366,
}


// uses the same keys!
const customWidth = {
  xs: 300,
  sm: 740,
  md: 1200,
}


export default () => (
  <Provider theme={theme}>
    {/* this container has default configuration */}
    <Container>
      <Row>...</Row>
      <Row>...</Row>
    </Container>


    {/* this container has overriden number of columns and its width */}
    <Container columns={7} width={customWidth}>
      <Row>...</Row>
      <Row>...</Row>
    </Container>
  </Provider>
)
You can be even more specific and define columns for each breakpoint independently or use callback for defining Container width.
import { Provider, Container, Row, Col, theme } from '@vitus-labs/coolgrid'


// defined breakpoints in Provider
const breakpoints = {
  xs: 320,
  sm: 768,
  md: 1366,
}


const customColumns = {
  xs: 4,
  sm: 6,
  md: 12,
}


// uses the same keys!
const customWidth = (defaultWidth) => ({
  ...defaultWidth,
  md: 1400,
})


export default () => (
  <Provider theme={theme}>
    {/* this container has overriden number of columns and its width */}
    <Container columns={customColumns} width={customWidth}>
      <Row>...</Row>
      <Row>...</Row>
    </Container>
  </Provider>
)

Properties

NameTypeDefault valueDescription
childrenReactNode-children to be rendered
widthobject | callback-defines container width for all breakpoints
columnsnumber | object | array-total number of possible columns in a row
sizenumber | object | array-size of Cols inside Container
gapnumber | object | array-spacing between Cols inside Container
paddingnumber | object | array-padding in all Cols inside Container
gutternumber | object | array-horizontal spacing of all Rows inside Container
contentAlignXstring | object | arrray-defines how should be Cols aligned inside Row on axis X
cssstring | func-property for extending Container styles (be careful, it override default styles like margins or paddings)
colCssstring | func-property for extending Cols styles within Container (be careful, it override default styles like margins or paddings)
rowCssstring | func-property for extending Rows styles within Container (be careful, it override default styles like margins or paddings)
componentstring | ComponentTypevalid HTML tag / component to be rendered instead of default component (renders div by default)
colComponentstring | ComponentType-valid HTML tag / component to be rendered instead of default component for all Cols inside Container (div by default)
rowComponentstring | ComponentType-valid HTML tag / component to be rendered instead of default component for all Rows inside Container (div by default)

Using props example

Simple value props

In the following example you can see how to simply use Container props.
import { Container } from '@vitus-labs/coolgrid'


export default () => (
  <Container
    columns={12}
    width={{ xs: '100%', sm: 540, md: 720, lg: 960, xl: 1140 }}
    size={4}
    gap={8}
    padding={8}
    gutter={0}
    contentAlignX="center"
    css={`
      text-align: center;
    `}
    colCss={`
      background-color: white;
    `}
    rowCss={`
      background-color: white;
    `}
    component="div"
    colComponent="div"
    rowComponent="div"
  >
    {/* ...some content */}
  </Container>
)

Responsive object type props

import { Container } from '@vitus-labs/coolgrid'


export default () => (
  <Container
    columns={{ xs: 12, sm: 6, md: 12 }}
    size={{ xs: 12, sm: 3, md: 3 }}
    gap={{ xs: 12, sm: 16, md: 24, lg: 32 }}
    padding={{ xs: 12, sm: 16, md: 24, lg: 32 }}
    gutter={{ xs: 0, sm: 16 }}
    contentAlignX={{ xs: 'center', md: 'left' }}
    css={{
      xs: `
      text-align: center;
    `,
      sm: `text-align: left;`,
    }}
  >
    {/* ...some content */}
  </Container>
)

Responsive array type props

import { Container } from '@vitus-labs/coolgrid'


export default () => (
  <Container
    columns={[12, 6, 12]} {/* as [xs, sm, md] */}
    size={[12, 3, 3]}
    gap={[12, 16, 24, 32]}
    padding={[12, 16, 24, 32]}
    gutter={[0, 16]}
    contentAlignX={['center', 'left']}
  >
    {/* ...some content */}
  </Container>
)

Props passed via Context

It's important to know what properties are passed to children components.

Props passed to Row

Following properties are the ones which are being used by Row component under the hood when defined globally on Container component.
  • gap
  • gutter
  • contentAlignX
  • rowComponent
  • rowCss

Props passed to Col

Following properties are the ones which are being used by Col component under the hood when defined globally on Container component.
  • columns
  • size
  • gap
  • gutter
  • padding
  • colComponent
  • colCss
