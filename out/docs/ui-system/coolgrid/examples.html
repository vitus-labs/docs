Examples

Custom columns grid

Property columns can be set either on Container or Row component.
// import { Container, Row, Col, Provider, theme } from '@vitus-labs/coolgrid'


render(
  <Provider theme={theme}>
    <Container padding={40} gap={2} colCss={defaultCss}>
      <Row>
        <Col size={1} />
        <Col size={1} />
        <Col size={1} />
        <Col size={1} />
        <Col size={1} />
        <Col size={1} />
        <Col size={1} />
        <Col size={1} />
        <Col size={1} />
        <Col size={1} />
        <Col size={1} />
        <Col size={1} />
      </Row>
      {/* changes total count of columnson row */}
      <Row columns={7}>
        <Col size={1} />
        <Col size={1} />
        <Col size={1} />
        <Col size={1} />
        <Col size={1} />
        <Col size={1} />
        <Col size={1} />
      </Row>
      {/* changes total count of columnson row */}
      <Row columns={5}>
        <Col size={1} />
        <Col size={1} />
        <Col size={1} />
        <Col size={1} />
        <Col size={1} />
      </Row>
    </Container>
  </Provider>
)

Equal column height

By default, all columns in a Row has equal height.
// import { Container, Row, Col, Provider, theme } from '@vitus-labs/coolgrid'


render(
  <Provider theme={theme}>
    <Container
      size={4}
      gap={2}
      padding={20}
      colCss={`
       ${defaultCss};
       white-space: pre-wrap;
       `}
    >
      <Row>
        <Col>
          <p>
            Lorem ipsum dolor sit amet, cibo sensibus interesset no sit. Et
            dolor possim volutpat qui. No malis tollit iriure eam, et vel tale
            zril blandit, rebum vidisse nostrum qui eu. No nostrud dolorem
            legendos mea, ea eum mucius oporteat platonem.Eam an case
            scribentur, ei clita causae cum, alia debet eu vel.
          </p>
        </Col>
        <Col />
        <Col />
      </Row>
    </Container>
  </Provider>
)

Nested columns

Columns and Rows can be also nested.
// import { Container, Row, Col, Provider, theme } from '@vitus-labs/coolgrid'


// components also reuse padding defined in Container
render(
  <Provider theme={theme}>
    <Container padding={20} gap={2} colCss={defaultCss}>
      <Row>
        <Col size={8}>
          <Row>
            <Col size={6}>col-6</Col>
            <Col size={6}>col-6</Col>
          </Row>
        </Col>
        <Col size={4} />
      </Row>
    </Container>
  </Provider>
)

One column with set size

We can define size only on some columns. The other columns will fit into the rest width.
// import { Container, Row, Col, Provider, theme } from '@vitus-labs/coolgrid'


render(
  <Provider theme={theme}>
    <Container padding={40} gap={2} colCss={defaultCss}>
      <Row>
        <Col />
        <Col size={5}>size 5</Col>
        <Col />
      </Row>
    </Container>
  </Provider>
)
