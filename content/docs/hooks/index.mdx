## Try It Live

<LiveCode code={`function HooksDemo() {
  const [count, setCount] = React.useState(0)
  const [debouncedCount, setDebouncedCount] = React.useState(0)
  const prevCount = React.useRef(0)

  // Simulated useDebounce
  React.useEffect(() => {
    const timer = setTimeout(() => setDebouncedCount(count), 300)
    return () => clearTimeout(timer)
  }, [count])

  // Simulated usePrevious
  React.useEffect(() => {
    prevCount.current = count
  })

  // Simulated useToggle
  const [isOn, setIsOn] = React.useState(false)

  return (
    <div style={{ fontFamily: 'system-ui', display: 'flex', flexDirection: 'column', gap: 16 }}>
      <div style={{ display: 'flex', gap: 8, alignItems: 'center' }}>
        <button onClick={() => setCount(c => c + 1)} style={{
          padding: '8px 16px', borderRadius: 6, border: '1px solid #ccc',
          cursor: 'pointer', fontSize: 14, background: '#fff',
        }}>
          Increment ({count})
        </button>
        <span style={{ fontSize: 13, color: '#666' }}>
          Previous: {prevCount.current} | Debounced: {debouncedCount}
        </span>
      </div>
      <div style={{ display: 'flex', gap: 8, alignItems: 'center' }}>
        <button onClick={() => setIsOn(v => !v)} style={{
          padding: '8px 16px', borderRadius: 6, border: 'none', cursor: 'pointer',
          fontSize: 14, fontWeight: 500,
          background: isOn ? '#198754' : '#dc3545', color: '#fff',
          transition: 'background 0.2s',
        }}>
          {isOn ? 'ON' : 'OFF'}
        </button>
        <span style={{ fontSize: 13, color: '#666' }}>useToggle pattern</span>
      </div>
    </div>
  )
}

render(<HooksDemo />)
`} noInline />

## Hook Categories

### State Management

| Hook | Description |
|------|-------------|
| [`useControllableState`](/docs/hooks/state#usecontrollablestate) | Unified controlled/uncontrolled pattern |
| [`useToggle`](/docs/hooks/state#usetoggle) | Boolean toggle with convenience methods |
| [`usePrevious`](/docs/hooks/state#useprevious) | Previous render value |
| [`useLatest`](/docs/hooks/state#uselatest) | Always-current value ref |

### DOM & Measurement

| Hook | Description |
|------|-------------|
| [`useElementSize`](/docs/hooks/dom#useelementsize) | Track element dimensions via ResizeObserver |
| [`useIntersection`](/docs/hooks/dom#useintersection) | Observe element visibility |
| [`useClickOutside`](/docs/hooks/dom#useclickoutside) | Detect clicks outside an element |
| [`useMergedRef`](/docs/hooks/dom#usemergedref) | Combine multiple refs into one |
| [`useScrollLock`](/docs/hooks/dom#usescrolllock) | Lock page scroll |
| [`useWindowResize`](/docs/hooks/dom#usewindowresize) | Track viewport dimensions |

### Events & Interaction

| Hook | Description |
|------|-------------|
| [`useHover`](/docs/hooks/events#usehover) | Mouse hover state |
| [`useFocus`](/docs/hooks/events#usefocus) | Focus/blur state |
| [`useFocusTrap`](/docs/hooks/events#usefocustrap) | Trap keyboard focus in container |
| [`useKeyboard`](/docs/hooks/events#usekeyboard) | Listen for key presses |

### Timing

| Hook | Description |
|------|-------------|

| [`useThrottledCallback`](/docs/hooks/timing#usethrottledcallback) | Throttle function calls |
| [`useInterval`](/docs/hooks/timing#useinterval) | Declarative setInterval |
