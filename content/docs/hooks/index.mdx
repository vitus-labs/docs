A comprehensive collection of 27 React hooks covering UI state management, DOM interactions, event handling, timing, theming, and accessibility.

## Installation

```package-install
@vitus-labs/hooks
```

## Hook Categories

### State Management

| Hook | Description |
|------|-------------|
| [`useControllableState`](/docs/hooks/state#usecontrollablestate) | Unified controlled/uncontrolled pattern |
| [`useToggle`](/docs/hooks/state#usetoggle) | Boolean toggle with convenience methods |
| [`usePrevious`](/docs/hooks/state#useprevious) | Previous render value |
| [`useLatest`](/docs/hooks/state#uselatest) | Always-current value ref |

### DOM & Measurement

| Hook | Description |
|------|-------------|
| [`useElementSize`](/docs/hooks/dom#useelementsize) | Track element dimensions via ResizeObserver |
| [`useIntersection`](/docs/hooks/dom#useintersection) | Observe element visibility |
| [`useClickOutside`](/docs/hooks/dom#useclickoutside) | Detect clicks outside an element |
| [`useMergedRef`](/docs/hooks/dom#usemergedref) | Combine multiple refs into one |
| [`useScrollLock`](/docs/hooks/dom#usescrolllock) | Lock page scroll |
| [`useWindowResize`](/docs/hooks/dom#usewindowresize) | Track viewport dimensions |
