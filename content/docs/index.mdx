Vitus Labs is a collection of packages spanning two areas: a **UI system** for building React interfaces, and **developer tools** for building, testing, linting, and shipping packages.

## UI System

Components, styling, layout, hooks, and theming for React applications.

| Package | Description |
|---------|-------------|
| [@vitus-labs/core](/docs/core) | Configuration, initialization, and shared utilities |
| [@vitus-labs/styler](/docs/styler) | High-performance CSS-in-JS engine (~3KB gzipped) |
| [@vitus-labs/attrs](/docs/attrs) | Composable attribute factory for component configuration |
| [@vitus-labs/elements](/docs/elements) | Base UI primitives: Element, List, Text, Overlay, Portal |
| [@vitus-labs/rocketstyle](/docs/rocketstyle) | Advanced component styling with themes, pseudo-states, and dimensions |
| [@vitus-labs/unistyle](/docs/unistyle) | Universal responsive styling utilities (170+ CSS properties) |
| [@vitus-labs/coolgrid](/docs/coolgrid) | Responsive grid system with Container, Row, and Col |
| [@vitus-labs/hooks](/docs/hooks) | Collection of 28 React hooks for common patterns |
| [@vitus-labs/connectors](/docs/connectors) | CSS-in-JS engine adapters (styler, emotion, styled-components, native) |
| [@vitus-labs/kinetic](/docs/kinetic) | Declarative enter/leave animations and transitions |
| [@vitus-labs/kinetic-presets](/docs/kinetic-presets) | Ready-made animation presets (fade, slide, scale) |
| [@vitus-labs/rocketstories](/docs/rocketstories) | Auto-generated Storybook stories from rocketstyle components |

## Quick Start

Install the core UI packages:

```package-install
@vitus-labs/core @vitus-labs/styler @vitus-labs/connector-styler @vitus-labs/elements @vitus-labs/rocketstyle
```

Initialize the CSS-in-JS engine in your app entry point:

```tsx
import { init } from '@vitus-labs/core'
import connector from '@vitus-labs/connector-styler'

init({ ...connector, component: 'div', textComponent: 'span' })
```

Then start building components:

```tsx
import { Element } from '@vitus-labs/elements'

function App() {
  return (
    <Element tag="main" padding={24}>
      Hello, Vitus Labs!
    </Element>
  )
}
```
