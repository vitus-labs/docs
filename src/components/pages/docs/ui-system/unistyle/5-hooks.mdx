# Hooks

The package has only one hook at this moment called `useWindowResize`.

## useWindowResize

`useWindowResize` is a React hook listening to window changes and returns
client `width` and `height`.

```tsx editor=false live=false
import { useWindowResize } from '@vitus-labs/unistyle'

// usage
const component = () => {
  const { width, height } = useWindowResize()

  return null
}

// options for customization
const component = () => {
  const { width, height } = useWindowResize(100, { width: 0, height: 0 })

  return null
}
```

### Properties

| Name                         | Type       | Default value           | Description                                     |
| ---------------------------- | ---------- | ----------------------- | ----------------------------------------------- |
| throttleDelay `(arg[0])`     | **number** | 200                     | a throttling delay value for better performance |
| { width, height } `(arg[1])` | **object** | { width: 0, height: 0 } | default `width` and `height` value              |
