# Text

Text is a basic component to simply render text content. By default, it
has reseted some styling.

## Examples

```tsx
// import { Text } from '@vitus-labs/elements'

// ↓↓↓ Text ↓↓↓
render(
  <>
    <Text label="This is a label" style={{ color: '#fff' }} />
    <Text style={{ color: '#fff' }}>This is a children content</Text>
  </>
)
```

## Using Text component

There are several ways of rendering text within Text component.
Let's take a look at the examples below.

### Children property

First of all, **Text** can be used and pass `children` prop
to render content.

```tsx
// import { Text } from '@vitus-labs/elements'

// ↓↓↓ Text ↓↓↓
render(<Text style={{ color: '#fff' }}>This is a children content</Text>)
```

### Label property

For shorter texts, or inline texts sometimes it could me more convenient
to use inline prop. For such a way, `label` prop could be used. In case
of passing both `children` and `label`, then **children** will have
a priority and `label` will be ignored.

```tsx
// import { Text } from '@vitus-labs/elements'

// ↓↓↓ Text ↓↓↓
render(<Text label="This is a label" style={{ color: '#fff' }} />)
```

## Text types

## Paragraph

By default it renders `span` element as an **inline** text element.
If you would like to render Text component as **block** element (paragraph),
you can set prop `paragraph` to **true**, or alternatively set property `tag`.

In case of using `paragraph` and `tag`, **tag** will have a priority over
paragraph prop.

```tsx
// import { Text } from '@vitus-labs/elements'

// ↓↓↓ Text ↓↓↓
render(
  <Text paragraph style={{ color: '#fff' }}>
    This is a block text which can have as children another
    <Text tag="strong">text component</Text>.
  </Text>
)
```

## Props

These are props to which can be passed to **Text**

| Name      | Type               | Default value | Description                                                                                       |
| --------- | ------------------ | :-----------: | :------------------------------------------------------------------------------------------------ |
| children  | **ReactNode**      |               | Children to be rendered within **Text** component.                                                |
| label     | **ReactNode**      |               | Label can be used instead of children for inline syntax. But **children** prop takes a precedence |
| paragraph | **boolean**        |               | Defines whether should behave as a block text element. Automatically adds **p** HTML tag          |
| tag       | **HTML valid tag** |               | Defines what kind of HTML tag should be rendered                                                  |
| css       | **ExtendCss**      |               | Defines what kind of HTML tag should be rendered                                                  |
